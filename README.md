# Java-Multithreaded-Chat-App

A robust Client-Server chat system built using Java Socket Programming and Multithreading. This project demonstrates how to handle multiple concurrent client connections to a single server for real-time communication.

## Quick Start & Developer Info

```bash
# 1. Compile both files
javac Server.java Client.java

# 2. Start the Server
java Server

# 3. Start the Client (in a new terminal)
java Client
